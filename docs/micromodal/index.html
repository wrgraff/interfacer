Modals: 2 types

I decided use fancy plugin Micromodal for modal windows. It's small, simple and a11y.

Modal

Classic modal window fades on all page. Very simple for use: add to button data-modal="elementId" and create this markup for window:

                        
                            <div class="modal" id="elementId" aria-hidden="true">
                                <section class="modal__overlay" tabindex="-1" data-modal-close>
                                    <div class="modal__container" role="dialog" aria-modal="true" aria-labelledby="elementIdTitle">
                                        <header class="modal__header">
                                            <h2 class="modal__title" id="elementIdTitle">Заголовок модального окна</h2>
                                        </header>

                                        <div class="modal__body">
                                            <!-- Body content -->
                                        </div>

                                        <footer class="modal__footer">
                                            <!-- Footer content -->
                                        </footer>

                                        <div class="modal__close">
                                            <button type="button" class="ico-button" data-modal-close>
                                                <span class="ico"><img src="/img/ico/baseline-close-24px.svg" alt="" /></span>
                                            </button>
                                        </div>
                                    </form>
                                </section>
                            </div>

Aside

Interfacer has aside panel: it works like modal window with other styles. Links and buttons creates the same way.

Markup for aside modal:

                        
                            <aside class="page-modal page-layout__modal" id="testAsideContent" aria-hidden="true">
                                <section class="page-modal__overlay" tabindex="-1" data-modal-close>
                                    <form action="#" class="page-modal__container" role="dialog" aria-modal="true" aria-labelledby="testContentTitle">
                                        <header class="page-modal__header">
                                            <h2 class="page-modal__title" id="testContentTitle">Заголовок модального окна</h2>
                                        </header>

                                        <div class="page-modal__body">
                                            <!-- Body content -->
                                        </div>

                                        <footer class="page-modal__footer">
                                            <!-- Footer content -->
                                        </footer>
                                    </form>
                                </section>
                            </aside>
